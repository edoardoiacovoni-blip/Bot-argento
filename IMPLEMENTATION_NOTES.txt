BOT ARGENTO - INTERACTIVE MODE IMPLEMENTATION
================================================================================

PROBLEM STATEMENT (Italian):
"Mi puoi dare un passo alla volta ....e aspettare che ti confermi di averlo 
fatto prima di darmi successivo?"

Translation: "Can you give me one step at a time... and wait for me to confirm 
that I've done it before giving me the next one?"

SOLUTION:
Implemented a step-by-step interactive mode that allows users to control the 
bot's execution by confirming each operation before it proceeds.

================================================================================
IMPLEMENTATION DETAILS
================================================================================

1. INTERACTIVE MODE FLAG
   - Environment variable: INTERACTIVE_MODE
   - Values: "true" (interactive) / "false" (automatic, default)
   - Easy to toggle between modes

2. CONFIRMATION FUNCTION
   - Function: wait_for_confirmation(step_description)
   - Prompts user before each critical operation
   - User commands:
     * ENTER: Confirm and proceed
     * 's': Skip current step
     * 'q': Quit program gracefully

3. CONFIRMATION POINTS
   The bot asks for confirmation before:
   - Checking institutional signals
   - Fetching market data and calculating opportunities
   - Executing each micro-trade
   - Converting profits to PAXG
   - Final checks and accumulation
   - Waiting before next cycle

4. ITALIAN LANGUAGE INTERFACE
   All prompts and messages are in Italian to match user's language preference

5. BACKWARD COMPATIBILITY
   - Default mode is automatic (no breaking changes)
   - Existing functionality preserved
   - Interactive mode is opt-in
